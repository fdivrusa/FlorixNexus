Frequently Asked Questions

Everything you need to know before we start working together.

Do you work with fixed prices or hourly rates? ▼
It depends on the project. For well-defined scopes (like showcase sites), we prefer fixed prices (packages). For complex, evolving software development, an hourly rate or daily rate ensures flexibility and quality.
Do we own the source code? ▼
Yes, absolutely. Once the project is fully paid for, you own 100% of the Intellectual Property and source code. We can also help you set up your own repositories.
Can you maintain existing .NET applications? ▼
Yes. We specialize in the Microsoft ecosystem. We can audit, fix bugs, optimize performance, or migrate legacy .NET applications to modern .NET Core / Cloud architectures.
